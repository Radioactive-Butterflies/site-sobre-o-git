Passos para instalar Git:
1- Acessar https://git-scm.com/ e selecione o download de acordo com o software desejado. (baixargit.png)
2- O download deverá começar sozinho, caso não comece, selecione para baixar manualmente. 
3- Apos o download concluido, execute o programa e clique na opção next. (instalaçãogit.png)
4- Selecione a priemira opção e install (instalaçãogit2.png)
5- O programa ira baixar e assim que concluir, marque as opçoes de acordo com a imagem abaixo.(instalaçãofinalizada.png)

Passos para login Github:
Entre em https://github.com/
Conta existente. 
1- Clique em Sign in (jacadastro.png)
2- Digite seu email ou usuario e senha (login.png)
3- Pagina aberta (loginfeito.png)
Conta Nova 
1- Clique em Sign up for GitHub(novaconta.png)
2- Digite seu email (email.png)
3- Crie uma senha (criarsenha.png)
4- Crie um nome de usuario (nomeusuario.png)
5- Caso deseje receber novidades e anuncios no email digite "y", caso contrario digite "n" (receberemail.com)
6- Faça a verificação solicitada (verificação .png) (verificaçãofeita.png)
7- Um codigo de verificação sera enviado a seu email. Digite o codigo no campo disponivel (verificaçãoemail.png)
8- Selecione o tipo de conta gratuita, ou clique em "Skip personalization" se quiser pular a personalização de sua conta no GitHub (selecionartipodeconta.png)
9-  Caso continue com a personalização, selecione quantos membros estarão trabalhando com voce, caso sua conta seja feita para um trabalho coletivo. Selecione tambem se você é aluno ou professor para utilizar o GitHub. (personalização.png)
10- Selecione quais opções sao interessantes para seu GitHub personalidade (personalização2.png)
11- Cadastro realizado (cadastrofeito.png)

Passos para login GitLab:
Acesse o link https://gitlab.com/users/sign_in
Login pelo GitHub:
1- Clique em GitHub (logingithub.png)
2- Autorize a conexão (conectar.png) 
3- Autorize os termos (aceitarostermos.png)
4- Login Feito (loginfeito.png)
Nova conta 
1- Clique em Register now para fazer uma nova conta (novologin.png)
2- Preencha os campos necessarios e clique em Register (cadastro.png)
3- Verifique seu email, clicando no link na plataforma do email que utiliza (verificar.png)
4- Selecione os campos de acordo com suas informaçoes e as informaçoes desejadas (parte1.png) (parte2.png) (parte3) 
5- Realize o login com seu email/usuario e senha criados (login.png)
6- Clique em "Ok, lte's go" (letsgo.png)
7- Login realizado (loginfeito.png)
