Passos para instalar Git:
1- Acessar https://git-scm.com/ e selecione o download de acordo com o software desejado. (baixargit.jpeg)
2- O download deverá começar sozinho, caso não comece, selecione para baixar manualmente. 
3- Apos o download concluido, execute o programa e clique na opção next. (instalaçãogit.jpeg)
4- Selecione a priemira opção e install (instalaçãogit2.jpeg)
5- O programa ira baixar e assim que concluir, marque as opçoes de acordo com a imagem abaixo.(instalaçãofinalizada.jpeg)

Passos para login Github:
Entre em https://github.com/
Conta existente. 
1- Clique em Sign in (jacadastro.jpeg)
2- Digite seu email ou usuario e senha (login.jpeg)
3- Pagina aberta (loginfeito.jpeg)
Conta Nova 
1- Clique em Sign up for GitHub(novaconta.jpeg)
2- Digite seu email (email.jpeg)
3- Crie uma senha (criarsenha.jpeg)
4- Crie um nome de usuario (nomeusuario.jpeg)
5- Caso deseje receber novidades e anuncios no email digite "y", caso contrario digite "n" (receberemail.jpeg)
6- Faça a verificação solicitada (verificação.jpeg) (verificaçãofeita.jpeg)
7- Um codigo de verificação sera enviado a seu email. Digite o codigo no campo disponivel (verificaçãoemail.jpeg)
8- Selecione o tipo de conta gratuita, ou clique em "Skip personalization" se quiser pular a personalização de sua conta no GitHub (selecionartipodeconta.jpeg)
9-  Caso continue com a personalização, selecione quantos membros estarão trabalhando com voce, caso sua conta seja feita para um trabalho coletivo. Selecione tambem se você é aluno ou professor para utilizar o GitHub. (personalização.jpeg)
10- Selecione quais opções sao interessantes para seu GitHub personalidade (personalização2.jpeg)
11- Cadastro realizado (cadastrofeito.jpeg)

Passos para login GitLab:
Acesse o link https://gitlab.com/users/sign_in
Login pelo GitHub:
1- Clique em GitHub (logingithub.jpeg)
2- Autorize a conexão (conectar.jpeg) 
3- Autorize os termos (aceitarostermos.jpeg)
4- Login Feito (loginfeito.jpeg)
Nova conta 
1- Clique em Register now para fazer uma nova conta (novologin.jpeg)
2- Preencha os campos necessarios e clique em Register (cadastro.jpeg)
3- Verifique seu email, clicando no link na plataforma do email que utiliza (verificar.jpeg)
4- Selecione os campos de acordo com suas informaçoes e as informaçoes desejadas (parte1.jpeg) (parte2.jpeg) (parte3.jpeg) 
5- Realize o login com seu email/usuario e senha criados (login.jpeg)
6- Clique em "Ok, lte's go" (letsgo.jpeg)
7- Login realizado (loginfeito.jpeg)
